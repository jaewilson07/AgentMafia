# Workflow Builder
Workflow Builder is a no-code way to build workflows, right in Slack. Any user can combine a limited set of steps and triggers to quickly set up an automation.
![](https://api.slack.com/marketing/img/feature/workflow-builder/hero-wfb-lp@2x.jpg)
## Workflow Builder for developers
Extend the functionality of Workflow Builder with your own workflow apps. Workflow Builder is a paid feature. For the most up-to-date information about pricing, take a look at our [pricing page](https://slack.com/help/articles/15363357403411/).
